Detailed Description

VCompositor 合成器

合成器是一种渲染器，主要用来做后期特效，目前功能较薄弱

Member Function Documentation

virtual void VCompositor::BindSketch ( VViewportPtr  viewport)
overridevirtual

绑定内部SketchBook，如果视口和内部SketchBook对不上的话会对内部SketchBook进行调整以适应视口， 如果存在前一个渲染器同时前一个渲染器的内部SketchBook与当前渲染器的SketchBook相同时不会绑定SketchBook 其余情况下都会绑定SketchBook

Reimplemented from VRendererBase.

virtual void VCompositor::UnbindSketch ( )
overridevirtual

如果存在后一个渲染器，且后一个渲染器的SketchBook与当前SketchBook相同时不会解绑定 其余情况下都会解绑定SketchBook 因此如果要让后一个渲染器和前一个渲染器共用SketchBook，那么只要将前一渲染器的SketchBook赋值给后一个渲染器即可
