3dMaster

The 3dMaster is a 3d scene viewer that supports 20+ 3d formats.

It is built using a modern render pipeline with the Vulkan API
based on the PBR (Physics-Based Rendering) material system, supports CSM (Cascaded Shadow Maps) shadow technology,
uses OIT (Order-independent transparency) technology to optimize the sorting of transparent objects,
both improved rendering quality and performance.
In particular, 3dMaster fully supports the skeletal animation system.
after opening the main scene, animations can be imported in separate and switched freely.
animation artists can preview their works on the iPhone.

The supported 3d scene formats include the following:

• obj
• fbx
• gltf
• stl
• collada
• ply
• blend
• 3ds
• lwo
• lws
• m3d
• md2
• md3
• mdl
• x3d
• x
• bvh
• ac
• b3d
• opengex
• dxf
• xgl

The supported texture formats include the following:

• ktx
• pvr
• dds
• jpeg
• png
• tga
• pcx
• psd
• gif
• bmp
• ppm
• pgm
• hdr
