Welcome to Block Puzzle in Space!

FullBody Poser is a realistic style poser-like app, It is based on the self-developed graphics engine called VEngine2 and uses a technique called FullBody IK. This technology makes digital characters even more special, You can drag an effector on a character's body, and every joint will be resolved to fit your dragging operation, unlike most other poser-like apps that only rotate one joint at a time. After using this technology, the character looks like a real puppet, which helps create a more delicate full-body pose and is also more fun.
FullBody Poser supports placing two characters in one scene at the same time. you can create a more dramatic story through the two characters. moreover, It supports snapping the current scene to an image, and you can use this function to create comics. soon I will be adding advanced features such as keyframe animation, pose reconstruction and estimation through camera, rigging user characters, so stay tuned.
