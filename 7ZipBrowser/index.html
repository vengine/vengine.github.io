7Zip Browser

7Zip 压缩包浏览器 是一个预览压缩包的工具。 它支持包括7z、Rar、Tar、BZip2在内的超过20种压缩格式。 而且它支持四种预览视图，就像是OSX的Finder一样。 你可以查看压缩包内的文件结构和文件信息。 特别是当文件是图片时，你可以查看整张图片而不需要解压压缩包

7Zip 壓縮檔瀏覽器 是一個預覽壓縮檔的工具。 它支持包括7z、Rar、Tar、BZip2在內的超過20種壓縮格式。 而且它支持四種預覽視圖，就像是OSX的Finder一樣。 你可以查看壓縮檔內的文件結構和文件信息。 特別是當文件是圖片時，你可以查看整張圖片而不需要解壓壓縮檔。

7Zip Browser is a Archive Preview utility. It Supported included 7z, Rar, Zip, Tar, BZip2 more than 20 kinds of Archive Formats. And It Supported 4 kinds of Views, like the OSX Finder. You can review the Hierarchy and the file information in Archive. Special, When the file is picture, you can review complete picture, but didn’t need decompress the archive.

7zipをブラウザがアーカイブプレビューユーティリティです。 サポートされている、それが7Z、Rar、Zip、Tar、Bzip2を含み、アーカイブ形式の20種類以上。 そして、それはOSXのFinderのようなビューの4種類を、サポートされている。 あなたは、階層やアーカイブ内のファイル情報を確認できます。 ファイルが画像である場合には、特別な、あなたは全体像を確認することができますが、アーカイブを解凍する必要はありませんでした。
